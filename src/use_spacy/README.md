# use_spacy

## Overview
これは自然言語処理ライブラリであるspacyの機能をまとめたものです。

参考にしたサイトは以下のとおりです。

https://ishitonton.hatenablog.com/entry/2018/11/24/004748

### このパッケージの機能は以下のようになっています。
- extract_entity.py

- split_sentence.py

→ 

- txt_pos.py

→ 

- txt_token.py
